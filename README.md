# Игра «Память»

Браузерная игра на запоминание карточек с тремя режимами игры и тремя уровнями сложности.

## Режимы игры

- **Обычный** — без ограничений, цель — найти все пары за минимальное время
- **С лимитом попыток** — ограниченное количество попыток
- **На время** — нужно успеть найти все пары до истечения таймера

## Уровни сложности

| Уровень | Карточек    | Попыток | Время |
|---------|-------------|---------|-------|
| Лёгкий  | 12 (6 пар)  | 24      | 2 мин |
| Средний | 16 (8 пар)  | 28      | 3 мин |
| Сложный | 24 (12 пар) | 36      | 3 мин |
